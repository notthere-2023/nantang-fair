关于这个网站

赶集会是什么

赶集会是南塘云村的一个定期公众集会，由富章发起和主持。从 2026 年 3 月到 6 月，每十天一次（10号、20号、30号），在 iMeeting 线上进行。自由讨论，民主决议——任何人都可以发言、提问、提出动议。

到第十期，赶集会已经走过了一个季度。这个网站整理了前九期的会议记录：谁来过、讨论了什么、决定了什么。

这个网站的来历

为第十期赶集会制作。送给富章哥的一份礼物——感谢他一季度的努力。

把散落在文件里的九期会议记录变成可以看、可以回溯的东西。看看这些信息被捞出来、放在一起之后，会生成什么新的意义。

也是一个学习的过程——尝试把生活中的问题转化为可编程的问题，在这个过程中学习技术。

想要呼应富章哥在第十期通知里关于合作的话：合作意味着什么？需要妥协吗？分歧时继续还是分手？

关于 AI

🤖 本网站的制作过程中使用了 AI 辅助：

· 数据提取——从会议纪要 Markdown 文件中提取结构化信息
· 代码编写——HTML / CSS / JavaScript 页面搭建
· 人物头像——由 DiceBear（开源头像库）生成

所有内容数据均来自赶集会会议记录原文。AI 处理的内容已经过人工审核。

技术

纯静态网页——HTML + CSS + 一点点 JavaScript。没有框架，没有后端，浏览器直接打开就能看。代码和数据可随时修改。
